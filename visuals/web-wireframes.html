What this is

Four web wireframes — the public-facing artifact (Public Profile) and the hiring-manager workflow (Search, Dashboard, Job Post). Combined with the desktop wireframes, this is the full v2.0 product surface.

Design system

Website light palette (warm paper #fafaf9, ink #080503, brand purple #8b5cf6). Instrument Serif display + Instrument Sans body + JetBrains Mono. Zero border-radius except pill CTAs. See 00-design-system.html.

Next

Web team scaffolds these in Next.js + Tailwind + shadcn/ui. The HM dashboard and Job Post UI are the v2.0 deliverables that unblock revenue. Public profile redesign is also v2.0.
