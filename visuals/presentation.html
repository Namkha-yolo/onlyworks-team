§ 07 · The technology

Boring. On purpose.

Layer
Choice
Why this, not that
Shell
Electron 33+
Tauri sounds cool but its rough edges (uiohook, auto-updater, signing) cost interns weeks. We need the bigger ecosystem.
Renderer
React + TypeScript + Vite
Boring, hireable, well-documented. TypeScript replaces tons of v1's defensive runtime code at compile time.
Styling
Tailwind v4 + shadcn/ui
v1 already had utility-class CSS — Tailwind is closer than it looks. shadcn is copy-pasteable, not a black box.
State
TanStack Query + Zustand
No Redux ever. React Query replaces v1's AppStore + manual cache invalidation entirely.
IPC
tRPC over IPC
Replaces 444-line preload.js with a typed bridge. Full TS inference. ~3 days to wire up.
Build
electron-vite
Hot reload in renderer = massive intern-velocity win. Keep electron-builder + electron-updater.
