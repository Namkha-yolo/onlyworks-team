Foundations

tokens · in :root

owButton() — single canonical system

12

Anti-patterns

do not reintroduce

✗ Banned class families

  • .btn-* / .intention-btn-* / .submit-btn
  • .permission-btn / .link-btn / .get-started-btn
  • .settings-*-btn / .session-action-btn
  • .share-report-btn / .manager-btn-*
  • .modal-overlay / .modal-content / .modal-header
  • .modal-body / .modal-actions / .close-btn

✗ Banned values

  • Hardcoded background: white inline (breaks dark mode)
  • Hardcoded #10b981, #22c55e, #ef4444, #dc2626 — use var(--success) / var(--error)
  • border-radius: 50 (missing %)
  • Any new design tokens added without explicit ask
  • Inline-rendered HTML without sanitizeHTML() on user data
